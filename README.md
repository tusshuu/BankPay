# 💳 BankPay – Banking Transaction Web Application

A responsive banking transaction web application built using **HTML, CSS, and JavaScript**. BankPay simulates real-world online banking operations such as user authentication, fund transfers, loan requests, account closure, and transaction history management through an interactive client-side interface.

## 🚀 Features

- 🔐 Secure user login with preloaded account credentials
- 💸 Transfer funds between user accounts
- 💰 Request and approve loans based on eligibility
- 📊 View transaction history and account summaries
- 📈 Display current balance, deposits, withdrawals, and interest
- ❌ Close account functionality
- 📱 Responsive and user-friendly interface
- ⚡ Dynamic UI updates using DOM manipulation

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)
- DOM Manipulation

## ▶️ Getting Started

### Clone the repository

```bash
git clone https://github.com/tusshuu/BankPay.git
```

### Navigate to the project

```bash
cd BankPay
```

### Run the project

Simply open **index.html** in your browser, or use the VS Code Live Server extension.

## 🔑 Demo Credentials

Use any of the preloaded accounts available in the application (refer to `sample-accounts.html` if included).
